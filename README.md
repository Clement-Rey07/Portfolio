# Portfolio
README.md

Ce document va vous permettre de vous repérer dans mon portfolio, étant donné qu'il est structuré d'une manière bien spécifique.
Ce que vous devez savoir : 
  - Les apprentissages critiques sont nommés : AC(num).(num)
  - Les Pièces justificatives sont nommées : PC-AC(num).(num)
  - Les Analyses réflexive sont nommées : AR-AC(num)-(num)
  - Chaque dossier d'apprentissage critique est composé d'un dossier de pièce justificative, d'un dossier d'analyse reflexive et d'une fiche d'évaluation de compétence respective a l'apprentissage critique étudié.
Enfin, tout les dossier sont rangés dans l'ordre numérique, ce qui veut dire qu'AC11.01 sera le premier dossier sur lequel vous aller tomber et que qu'il en va ainsi de suite.
