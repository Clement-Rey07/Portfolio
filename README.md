# Portfolio Symfony

## Description
Portfolio développé avec Symfony.
Il contient :
- une page d'accueil
- un formulaire d'inscription
- une génération de CV sans base de données

## Prérequis
- PHP >= 8.1
- Composer
- Git

## Installation
```bash
git clone https://github.com/TON_NOM/portfolio-symfony.git
cd portfolio-symfony
composer install
